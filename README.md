## &#10024; О книге

**«Свечи Силы»** — практическое руководство по работе с арканами Таро через свечную магию. Книга объединяет древние знания о четырёх мирах реальности, архетипах и энергетических узлах с практическими техниками свечных ритуалов.

### Три части книги:

| Часть | Название | Содержание |
|:-----:|:---------|:-----------|
| **I** | **Основания** | Устройство миров, природа души, архетипы |
| **II** | **Карта** | Десять узлов силы, 22 пути, стихии |
| **III** | **Практика** | Подготовка, ритуалы, решение проблем |

## &#128736;&#65039; Технологии

```
HTML5 + CSS3 + Vanilla JavaScript
```

- **Анимации** — чистый CSS (keyframes, transforms)
- **Рендеринг** — SVG для сакральной геометрии
- **Данные** — JSON для хранения контента
- **Шрифты** — Philosopher (заголовки), Cormorant Garamond (текст)

### Структура проекта

```
candle-book/
├── index.html          # Основной файл
├── book_data.json      # Контент книги (162,687 символов)
├── BOOK_STYLED.docx    # Книга для скачивания
└── README.md           # Этот файл
```

## &#128640; Развёртывание

### GitHub Pages

1. Fork этот репозиторий
2. Settings → Pages → Source: `main` branch
3. Сайт будет доступен по адресу `https://username.github.io/candle-book/`

### Локально

```bash
# Клонировать репозиторий
git clone https://github.com/jetmil/candle-book.git
cd candle-book

# Запустить локальный сервер (нужен для загрузки JSON)
python -m http.server 8000
# или
npx serve .

# Открыть http://localhost:8000
```
